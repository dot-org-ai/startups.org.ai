# Industries

Industries represent business sectors from the North American Industry Classification System (NAICS).

```typescript
import { Industry, IndustryVertical } from 'startups.org.ai'

const industry: Industry = {
  ns: 'industries.org.ai',
  type: 'Industry',
  id: 'HospitalsGeneral',
  name: 'General Medical and Surgical Hospitals',
  description: 'Establishments known and licensed as general medical and surgical hospitals primarily engaged in providing diagnostic and medical treatment...',
  code: '622110',
  shortName: 'Hospitals',
  sourceType: 'NAICS',
  level: 5
}
```

## NAICS Hierarchy

| Level | Digits | Description |
|-------|--------|-------------|
| 1 | 2 | Sector (e.g., 62 - Health Care) |
| 2 | 3 | Subsector (e.g., 622 - Hospitals) |
| 3 | 4 | Industry Group (e.g., 6221 - General Hospitals) |
| 4 | 5 | NAICS Industry (e.g., 62211 - General Medical) |
| 5 | 6 | National Industry (e.g., 622110 - General Med/Surg) |

## Filtering Industries

```typescript
const config: OntologyDimensionConfig = {
  enabled: true,
  priority: 6,
  filter: {
    // Filter by specific IDs
    ids: ['Healthcare', 'Finance', 'Legal'],
    // Or by hierarchy level
    levels: [1, 2, 3],  // Sector through industry group
    // Or by name
    namePattern: 'Health|Hospital|Medical',
    // Limit results
    limit: 100
  }
}
```
