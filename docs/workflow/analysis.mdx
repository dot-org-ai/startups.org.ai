# Analysis Phase

The analysis phase evaluates experiment results, extracts insights, and recommends next actions.

```typescript
import { analyzeResults } from 'startups.org.ai'

const analysis = await analyzeResults({
  experiment,
  metrics: [
    { type: 'visits', value: 1200 },
    { type: 'waitlist-signups', value: 36 },
    { type: 'conversion-rate', value: 3.0 },
    { type: 'time-on-site', value: 120 }
  ]
})
```

## Result Types

| Result | Meaning | Next Step |
|--------|---------|-----------|
| `winner` | Success criteria met | Advance concept |
| `loser` | Failure criteria met | Kill or pivot |
| `inconclusive` | Not enough data | Extend or rerun |
| `learning` | Insights but no clear winner | Apply learnings |

## Confidence Levels

| Confidence | Interpretation |
|------------|----------------|
| >95% | Very high confidence |
| 90-95% | High confidence |
| 80-90% | Moderate confidence |
| <80% | Low confidence, need more data |

## Iteration Loop

```typescript
const iterate = async (concept: StartupConcept) => {
  // Run experiment
  const experiment = await designExperiment({ concept })
  await runExperiment(experiment)

  // Analyze results
  const analysis = await analyzeResults({ experiment })

  if (analysis.result === 'winner') {
    // Advance to next phase
    concept.status = 'validated'
    return { action: 'build', concept }
  } else if (analysis.result === 'loser') {
    // Kill or pivot
    concept.status = 'invalidated'
    return { action: 'pivot', concept, insights: analysis.insights }
  } else {
    // Run next experiment
    const nextExp = await designExperiment({
      concept,
      experimentType: analysis.nextExperiment
    })
    return { action: 'iterate', experiment: nextExp }
  }
}
```
