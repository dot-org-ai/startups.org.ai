# Workflow

Workflows orchestrate the startup factory pipeline from hypothesis to validated startup. Each phase builds on the previous, transforming raw ideas into tested concepts.

```typescript
import { WorkflowExecution, executeWorkflow } from 'startups.org.ai'

const execution = await executeWorkflow({
  hypothesis: verticalAICFO,
  config: {
    hypothesisId: 'vertical-ai-cfos',
    ontologyFilters: {
      industryIds: ['healthcare', 'legal', 'manufacturing']
    },
    controls: {
      maxConcepts: 10,
      minViabilityScore: 70,
      includeVariants: true
    }
  },
  brandFamily: headlessLy,
  experimentBudget: 5000
})
```

## Pipeline Phases

```
ENRICH → GENERATE → SCORE → BRAND → PRODUCTIZE → DEPLOY → EXPERIMENT → ANALYZE
```

| Phase | Function | Output |
|-------|----------|--------|
| Enrichment | Add AI insights to ontology | Enriched entities |
| Generation | Cross-product hypothesis × ontology | Raw concepts |
| Scoring | Evaluate viability | Ranked concepts |
| Branding | Assign names and identity | Branded concepts |
| Productization | Package as services | Service definitions |
| Deployment | Generate and deploy assets | Live sites |
| Experimentation | Run validation experiments | Experiment data |
| Analysis | Analyze and recommend | Validated winners |
