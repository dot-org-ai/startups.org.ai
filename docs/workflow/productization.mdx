# Productization Phase

The productization phase packages branded concepts as productized services with clear scope, deliverables, pricing, and process.

```typescript
import { productizeService } from 'startups.org.ai'

const { service, landingPageCopy } = await productizeService({
  concept,
  pattern: 'audit',  // Service pattern to follow
  pricingTiers: 3
})
```

## Service Patterns

Choose a pattern that fits the concept:

| Pattern | Description | Example |
|---------|-------------|---------|
| `audit` | Review and assess | Financial Health Audit |
| `analysis` | Deep analysis | Cash Flow Analysis |
| `done-for-you` | Full service | Monthly Reporting Service |
| `managed` | Ongoing management | Managed Finance Operations |
| `coaching` | 1:1 guidance | CFO Coaching |

## Pattern Selection

```typescript
// Map concept characteristics to patterns
const selectPattern = (concept: StartupConcept): ServicePattern => {
  if (concept.serviceType === 'full-automation') return 'done-for-you'
  if (concept.aiOverlays.includes('coaching')) return 'coaching'
  if (concept.aiOverlays.includes('analysis')) return 'analysis'
  return 'audit'  // Default
}
```
