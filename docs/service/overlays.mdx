# AI Overlays

Even for tasks that can't be fully automated, AI can provide valuable overlay services—coaching, training, analysis, and more.

```typescript
import { AIOverlayType, AIOverlayTiming, CoachingOverlay } from 'startups.org.ai'

type OverlayType =
  // Pre-task
  | 'planning'        // Plan approach
  | 'preparation'     // Gather inputs
  | 'training'        // Teach skills
  | 'simulation'      // Practice first

  // During-task
  | 'coaching'        // Real-time guidance
  | 'assistance'      // AI helps with parts
  | 'co-pilot'        // Work alongside human
  | 'monitoring'      // Track progress

  // Post-task
  | 'analysis'        // Analyze outcomes
  | 'review'          // QA check
  | 'feedback'        // Improvement feedback
  | 'documentation'   // Document work

  // Meta
  | 'consulting'      // Strategic advice
  | 'optimization'    // Process improvement
  | 'benchmarking'    // Compare to best practices
  | 'forecasting'     // Predict outcomes
```

## Overlay Timing

| Timing | Description | Examples |
|--------|-------------|----------|
| `before` | Before the task | Planning, Training |
| `during` | During execution | Coaching, Co-pilot |
| `after` | After completion | Review, Analysis |
| `ongoing` | Continuous | Monitoring, Consulting |

## Overlay Types

### Coaching

Real-time guidance during task execution:

```typescript
const coaching: CoachingOverlay = {
  type: 'coaching',
  timing: 'during',
  capabilities: {
    realTimeGuidance: true,
    contextualTips: true,
    mistakePrevention: true,
    bestPractices: true,
    adaptiveAdvice: true
  },
  interaction: {
    mode: 'hybrid',
    channels: ['chat', 'overlay'],
    frequency: 'As needed'
  },
  valueProps: {
    headline: 'AI Coach for Financial Analysis',
    benefits: ['Reduce errors by 90%', 'Complete tasks 2x faster'],
    metrics: ['Error rate', 'Completion time', 'Quality score']
  }
}
```

## Overlay Bundles

Package multiple overlays for a task or process:

```typescript
const bundle: TaskOverlayBundle = {
  taskId: 'PrepareFinancialReports',
  taskName: 'Prepare Financial Reports',
  availableOverlays: ['planning', 'coaching', 'analysis', 'review'],
  overlays: {
    planning: planningConfig,
    coaching: coachingConfig,
    analysis: analysisConfig,
    review: reviewConfig
  },
  pricing: {
    individual: { planning: 99, coaching: 199, analysis: 149, review: 79 },
    bundle: 399,
    subscription: 299
  },
  totalValue: {
    timesSaved: '20 hours/month',
    errorReduction: '90%',
    qualityImprovement: '40%',
    roiEstimate: '10x'
  }
}
```
