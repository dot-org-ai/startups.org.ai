## Pricing Tiers

### Best Practices

**3 tiers is optimal** — Good, Better, Best

| Tier | Purpose | Typical Buyer |
|------|---------|---------------|
| Starter | Low barrier entry | Small business, testing |
| Pro | Most value, most buyers | Growth stage |
| Enterprise | Custom, high-touch | Large organizations |

### Tier Structure

```typescript
const tier: PricingTier = {
  name: 'Pro',

```

## Billing Periods

| Period | Use Case |
|--------|----------|
| `one-time` | Project-based services |
| `monthly` | Recurring services |
| `quarterly` | Seasonal services |
| `annual` | Committed relationships |

## Trust Signals

Build confidence in pricing:

| Element | Purpose |
|---------|---------|
| `freeTrialDays` | Risk-free testing |
| `moneyBackDays` | Purchase confidence |
| `customQuote` | Enterprise flexibility |

## Pricing Psychology

- **Anchor high** — Show enterprise tier first
- **Highlight middle** — Mark as "Most Popular"
- **Annual discount** — Incentivize commitment
- **Price on value** — Not on cost
